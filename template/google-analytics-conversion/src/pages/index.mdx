# Sample app

This application analyzes sample data about customer orders in retail.

## Sales by Product

Try setting the *Minimum Sales* to 5000.

export const minSales = variable("10000");

<Card className="p-6 mt-2">
    <label>Minimum Sales</label>
    <VariableInput variable={minSales} />
    <Embed
        height="600px"
        width="100%"
        alias={"visualize"}
        variables={{min_sales:minSales}}
    />
</Card>

## Orders

You can filter with **Product Name**.

export const productSelected = variable("all");
export const productNames = getJson({
    alias: "get_product_names"
})

<Grid cols="1">
    <VariableSelect variable={productSelected}>
        <VariableSelectItem value="all">All</VariableSelectItem>
        <VariableSelectItems items={productNames} labelKey="product_name" valueKey="product_name"/>
    </VariableSelect>
    <div>
        <DataTable
            alias={"get_all_orders"}
            variables={{
                product_name: productSelected
            }}
            height = {300}
        />
    </div>
</Grid>
