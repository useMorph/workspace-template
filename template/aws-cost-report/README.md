### Getting started

Write connection info in `~/.morph/config.yml` like this:

```yml
connections:
  connection_name:
    type: postgres
    host: localhost
    port: 5432
    user: postgres
    password: postgres
```

You can set it up by running the following command:

```bash
morph config
```
